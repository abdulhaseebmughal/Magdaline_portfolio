# Magdaline Mwihaki — Content Strategist & Positioning Expert

A professional portfolio and service platform for Magdaline Mwihaki, helping coaches and founders fix their message, positioning, and distribution.

## Tech Stack

- **Framework**: React with Vite
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI
- **Animations**: CSS Reveal + Intersection Observer
- **State Management**: TanStack Query (React Query)

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
